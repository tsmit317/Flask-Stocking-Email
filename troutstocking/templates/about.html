About

Typically the NC Wildlife Resources Commission releases an annual trout stocking schedule on their website, informing the public of when and what streams will be stocked. Unfortunately, COVID-19 threw a wrench in the stocking schedule this year. While NCWRC does post daily stocking updates online, it requires individuals to check the website everyday to see if their local/favorite streams have been stocked. Thus the idea to create an automated email notification was born.

How it works

Utilizing web scraping through pythons Beautifulsoup library, the application checks NCWRC's Daily Stocking Update webpage for any updates for the current day. If there have been any new stockings for the current day, the application queries its database to find users who have selected notifications for the counties which have been stocked. This information is then sent via E-mail to the user.
